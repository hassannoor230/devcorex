# Devcorex — Premium Software House Website

A luxury VIP website for Devcorex built with **React + Vite + Framer Motion**.

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🛠️ Tech Stack

- **React 18** — UI Framework
- **Vite 5** — Build Tool
- **Framer Motion** — Animations
- **Lucide React** — Icons
- **Google Fonts** — Cormorant Garamond + Outfit + Space Mono

## 📄 Sections

1. **Hero** — Particle canvas, animated headline, stats
2. **Tech Band** — Scrolling tech stack ticker
3. **About** — Company values, CEO quote
4. **Services** — 6 service cards with hover effects
5. **Portfolio** — Filterable project showcase
6. **Testimonials** — Client reviews carousel
7. **Team** — Featured CEO (Hassan Noor) + core team
8. **Contact** — Contact form + info
9. **Footer** — Links, social, branding
